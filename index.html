PyEndicia Documentation

This project came about from a failed project

This API access meets the requirements for using Endicia's API baed on their documentation and my conversations with their dev team and sales team.

It fulfills the following requirements:

  • Change Password
  • Buy Postage
  • Label

These are as fulfilled as best as I could with their limited development environment.

To extend the library is quite simple to do and requires you to write a new Builder, new Breaker, and a new Request class. Then you can very easily request this new API call from within the library.
